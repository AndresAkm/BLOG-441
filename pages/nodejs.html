Node.JS - desarrollo web

¿Qué es Node.JS?

Node.JS es un entorno de ejecución de Javascript basado en el motor V8 de Google para ejecutar codigo Javascript fuera del navegador y desde el lado del backend, basado en el paradigma programación asíncrona.

¿Cúal es la diferencia entre JavaScript en el navegador y en el frontend?

- JavaScript en el navegador

Es código que se ejecuta en el navegador del usuario (chrome, safari, firefox...). Tiene acceso directo al DOM, window, eventos, etc, todo se ejecuta del lado del cliente

- JavaScript en el frontend

Es todo JS que corre del lado del cliente, pero en formas diferentes al navegador, incluyendo apps, WebViews o frameworks como React, Vue o Angular.

! Todo JavaScript en el navegador es frontend, pero no todo frontend depende solo del navegador.

Ventajas y desventajas del uso de JavaScript en el backend

- Ventajas
  • Permite usar el mismo lenguaje en frontend y backend (full-stack JS)
  • Entorno rápido y eficiente gracias al motor V8
  • Gran ecosistema de librerías y paquetes (npm)
  • Ideal para apps en tiempo real con ayuda de WebSockets (chats, notificaciones, etc.)
- Desventajas
  • Menor rendimiento en tareas altamente computacionales.
  • La asincronía puede dificultar la depuración si no se gestiona bien.
  • Madurez menor en comparación con entornos como Java o .NET

💡 Node.js es potente para muchos casos, pero como toda herramienta, debe usarse donde más eficiente sea.

¿Cómo instalar Node.JS?

Para instalar Node.JS en tu sistema, tienes varias opciones, pero te mostraré la más sencilla para que puedas tener Node.JS instalado en tu computadora.

Descarga directa

Paso 1: Ve al sitio oficial de Node.js

Dirígete a https://nodejs.org/en/download y verás dos versiones disponibles:

  • LTS (Long Term Support): Versión estable recomendada para la mayoría de usuarios
  • Current: Versión más reciente con las últimas características

💡 Recomendación: Siempre elige la versión LTS para proyectos de producción.

Paso 2: Descarga el instalador

Haz clic en el botón de descarga correspondiente a tu sistema operativo:

  • Windows: Archivo .msi
  • macOS: Archivo .pkg
  • Linux: Paquete precompilado o código fuente

Paso 3: Ejecuta el instalador

Una vez descargado, ejecuta el archivo y sigue el asistente de instalación:

  • Acepta los términos y condiciones
  • Elige la ubicación de instalación (recomendado: ubicación por defecto)
  • Asegúrate de que esté marcada la opción "Add to PATH"
  • Completa la instalación

Paso 4: Comprobar la instalación

Una vez descargado, abriremos una nueva términal (cmd o powershell) y escribiremos los siguientes comandos.

node -v

Luego:

npm -v

Importar y Exportar Módulos en Node.js

Los módulos son una de las características más importantes de Node.js. Te permiten organizar tu código en archivos separados y reutilizar funciones, variables y objetos entre diferentes partes de tu aplicación.

¿Qué son los módulos?

Un módulo en Node.js es simplemente un archivo JavaScript que puede exportar funciones, objetos, variables o clases para ser utilizadas en otros archivos.

  • module.exports: Sintaxis tradicional de Node.js (CommonJS)
  • export/import: Sintaxis moderna de ES6 (ECMAScript Modules)

💡 Nota: En esta guía usaremos la sintaxis CommonJS que es la más común en Node.js

Ejemplo práctico: Calculadora modular

Vamos a crear una calculadora dividida en módulos para entender mejor el concepto.

Paso 1: Crea la estructura de carpetas

Primero, crea una carpeta llamada "calculadora-modular" y organiza los archivos (el archivo package.json es un archivo que abordaremos más adelante en el blog, pero puedes crearlo con el comando "npm init -y"):

calculadora-modular/
├── operaciones.js # Módulo con las funciones
├── calculadora.js # Archivo principal
└── package.json # Configuración del proyecto

Paso 2: Crea el módulo de operaciones (operaciones.js)

Este archivo contendrá las funciones matemáticas que queremos exportar:

// operaciones.js

// Función para sumar dos números
function sumar(a, b) {
    return a + b;
}

// Función para restar dos números
function restar(a, b) {
    return a - b;
}

// Función para multiplicar dos números
function multiplicar(a, b) {
    return a * b;
}

// Función para dividir dos números
function dividir(a, b) {
    if (b === 0) {
        throw new Error('No se puede dividir por cero');
    }
    return a / b;
}

// Variable constante
const PI = 3.14159;

// EXPORTAR las funciones y variables
module.exports = {
    sumar,
    restar,
    multiplicar,
    dividir,
    PI
};

Paso 3: Formas alternativas de exportar

Node.js te ofrece diferentes maneras de exportar. Aquí algunas alternativas:

Opción A: Exportar individual

// Al final de cada función
module.exports.sumar = sumar;
module.exports.restar = restar;
module.exports.PI = PI;

Opción B: Exportar directo

// Directamente en la declaración
exports.sumar = function(a, b) {
    return a + b;
};

Paso 4: Crea el archivo principal (calculadora.js)

Ahora vamos a importar y usar las funciones del módulo operaciones:

// calculadora.js

// IMPORTAR el módulo completo
const operaciones = require('./operaciones');

// Usar las funciones importadas
console.log('=== CALCULADORA MODULAR ===');
console.log(`Suma: 5 + 3 = ${operaciones.sumar(5, 3)}`);
console.log(`Resta: 10 - 4 = ${operaciones.restar(10, 4)}`);
console.log(`Multiplicación: 6 * 7 = ${operaciones.multiplicar(6, 7)}`);
console.log(`División: 15 / 3 = ${operaciones.dividir(15, 3)}`);
console.log(`Valor de PI: ${operaciones.PI}`);

Paso 5: Formas alternativas de importar

También puedes importar elementos específicos usando destructuring:

// Importar funciones específicas
const { sumar, restar, PI } = require('./operaciones');

// Ahora puedes usarlas directamente
console.log(`Suma: ${sumar(8, 2)}`);
console.log(`Resta: ${restar(10, 3)}`);
console.log(`PI: ${PI}`);

Paso 6: Ejecuta el programa

Guarda ambos archivos y ejecuta el programa desde la terminal:

cd calculadora-modular
node calculadora.js
